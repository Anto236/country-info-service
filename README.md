# country-info-service

Channel Developer Case Study - Country Information Service

## Project Details

- **Project Name:** country-info-service
- **Group ID:** com.ncba
- **Artifact ID:** country-info-service
- **Java:** 21
- **Spring Boot:** 3.4.4

## Run Locally

```bash
# Windows
mvnw.cmd spring-boot:run

# Linux / macOS
./mvnw spring-boot:run
```

Or with Maven installed:

```bash
mvn spring-boot:run
```

## Build

```bash
mvn clean package
```
